3.字符串三个作业

4.1 假定给定两个字符串（可以是空串）s1和s2,要将这两个串合并在一起，
要求是：s1串的第一个字符与s2串的第一个字符连接在一起，后续以此类推。
若一个串的长度大于另一个串的长度，则保留长串的剩余字符，即连接到新串的末尾即可。
例如：若s1=’abcd’、s2=’efghi’，则新串为：‘aebfcgdhi’.

源码：
s1=input(“请输入第一个字符串”)
s2=input(“请输入第二个字符串”)
#s1=”abcd”
#s2=”efghijk”
#用于测试的字符串
s3=””
if s1==””:
s3+=s2
elif s2==””:
s3+=s1
else:
while 1:
s3 += s1[0] + s2[0]
s1 = s1[1:]
s2 = s2[1:]
if s1==””:
s3 += s2
break
elif s2==””:
s3 += s1
break
print(“结果是”+s3)

4.2 对于用户输入的任意字符串，只显示字母和数字

源码：
s=input(“请输入一个字符串”)
r=””
for i in range(len(s)):
if s[i].isalnum()==True:#判断第i个是否是数字或者字母
r=r+s[i]
print(r)

4.3 制作随机验证码，不区分大小写。流程：
1）用户执行程序
2）给用户显示需要输入验证码
3）用户输入的值用户输入的值和显示的值相同时显示正确的信息:否则继续生成验证码等待用户输入

源码：
import random
while 1:
s2 = “”
for i in range(4):
n = random.randrange(0, 3) # 生成随机数0或1或2，用来判断下面，是生成随机数字，还是字母
if n == 0:
num = random.randrange(65, 91) # 为0时候，生成大写字母
s2 += chr(num)
elif n == 1:
num = random.randrange(97, 122) # 为1时候，生成小写字母
s2 += chr(num)
else:
k = random.randrange(0, 10) # 为1时候，生成数字
s2 += str(k)
print(“验证码是”+s2)
s1=input(“请输入验证码”)
if s1==s2:
print(“输入正确，结束”)
break
else:
s1 = input(“验证码错误，请重新输入”)
