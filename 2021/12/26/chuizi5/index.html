5.导师分配

Python读取<待分配学生模板buu.xlsx>，<待分配导师模板buu.xlsx>中的内容，
按照要求实现学生与导师之间的随机分配。随机分配的结果写入到<随机分配结果模板buu.xlsx>
注意：
1）<待分配学生模板buu.xlsx>，<待分配导师模板buu.xlsx>这两个文件中的行数并不固定。
2）一名学生对应一名导师，一名导师对应多名学生。
3）模板
附件1：待分配学生模板buu.xlsx
附件2：待分配导师模板buu.xlsx
附件3：随机分配结果模板buu.xlsx

源码：
from openpyxl import load_workbook
from openpyxl import Workbook
import openpyxl
import random

teachers={}#把老师作为一个字典
tutor= load_workbook(‘1_待分配导师模板buu(1)(1).xlsx’)#导入导师模板
sheetst= tutor.worksheets # 获取导师模板所有的sheet
tu= sheetst[0] # 把第一张sheet赋给“tu”

for i in range(2,8):#共有7名导师，第一行为表格，i作为行数，从2循环到8
teachers[tu.cell(i,1).value]=tu.cell(i,2).value
#把第i行第1列的数据（老师姓名）作为字典的key值，把第i行第2列的数据（老师的可分配学生数量）作为字典的value值
print(teachers)
#读取学生数据直接写到最后的表格里，再依次随机导师，随到的导师value减1
student=load_workbook(‘待分配学生模板buu.xlsx’)#导入学生模板
sheetss=student.worksheets #获取学生模板的所有sheet
su=sheetss[0] # 把第一张sheet赋给“su”

result=openpyxl.load_workbook(“随机分配结果模板buu.xlsx”)#导入结果模板
re=result.worksheets[0]#把结果模板的第一张sheet赋值给“re”
t = 0#t是一个flag，用来判定导师是否成功被分配
for i in range(2,38):
re.cell(i,1,su.cell(i,2).value)
re.cell(i,2,su.cell(i, 1).value)
re.cell(i,3,su.cell(i, 3).value)
#以下是随机分配老师部分
while 1:
num=random.randint(1,9)
for k in teachers:
if teachers[k] == num:
re.cell(i, 4, k)
print(k)
teachers[k] -= 1
print(teachers[k])
t=1
break
if t==1:
t=0
break

result.save(filename=”随机分配结果模板buu.xlsx”)
